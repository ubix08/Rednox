üìù How API Keys Work

1. Configuration: Set your API keys in this Settings tab. They're stored securely in your browser's localStorage.

2. Usage: When flows execute, the Worker reads API keys from environment variables: 

// In wrangler.toml
[vars]
GEMINI_API_KEY = "your-key-here"
OPENAI_API_KEY = "your-key-here"
ANTHROPIC_API_KEY = "your-key-here"

3. Security: API keys should be set as Worker secrets in production: 

npx wrangler secret put GEMINI_API_KEY
npx wrangler secret put OPENAI_API_KEY
npx wrangler secret put ANTHROPIC_API_KEY
