Getting Started

1. Initialize Database

Click "Init DB" to create the necessary database tables.

2. Create Your First Flow

Click "New Flow" and provide the flow configuration as JSON.

3. Test Your Flow

Each flow gets endpoints based on http-in nodes. Test them using curl or Postman.

Example Flow Structure

{
  "id": "my-chatbot",
  "name": "My Chatbot",
  "description": "A simple chatbot",
  "nodes": [
    {
      "id": "http-in-1",
      "type": "http-in",
      "url": "/chat",
      "method": "post",
      "wires": [["function-1"]]
    },
    {
      "id": "function-1",
      "type": "function",
      "func": "msg.payload = { response: 'Hello!' }; return msg;",
      "wires": [["http-out-1"]]
    },
    {
      "id": "http-out-1",
      "type": "http-response",
      "wires": []
    }
  ]
}

Create Flow

The JSON must include: id, name, description (optional), and nodes array. The id and name will be extracted from the JSON.
